# Soluzio - Website

Landing page for Soluzio technology solutions.

## Structure
- `index.html` - Main HTML file with SEO and accessibility improvements
- `styles.css` - All CSS styles with modern CSS variables
- `script.js` - All JavaScript functionality including form validation

## Features

### SEO Optimization
- Complete meta tags (description, keywords, author, robots)
- Open Graph tags for social media sharing
- Twitter Card tags
- Schema.org JSON-LD structured data
- Canonical URL

### Accessibility
- WCAG AA compliant color contrast
- Proper form labels with ARIA attributes
- Semantic HTML5 structure
- ARIA roles and labels
- Keyboard navigation support

### Performance
- Critical CSS inline for above-the-fold content
- Async CSS loading
- Deferred JavaScript loading
- Resource preconnect for Formspree
- Minimal dependencies (~30KB total)

### Modern Features
- CSS custom properties (variables)
- Client-side form validation
- Responsive design (mobile, tablet, desktop)
- Dynamic copyright year
- Smooth scrolling
- Intersection Observer animations

## Deployment
Hosted on Cloudflare Pages with auto-deploy from GitHub `main` branch.

## Local Development
Simply open `index.html` in a modern browser.

No build process required - this is a zero-dependency static site.
